# Data links subsystem — L2 requirements

## L2-LNK-001

- **Requirement:** Tranquility shall implement `GET /api/links/{instance}` with documented link metadata fields.
- **Rationale:** Link visibility is required for mission operations dashboards and automation.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Test
- **Parent:** L1-LNK-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN an instance with configured links WHEN list-links endpoint is queried THEN each link includes name, disabled status, counters, and detail fields.

## L2-LNK-002

- **Requirement:** Tranquility shall support documented link enable, disable, and counter-reset operations.
- **Rationale:** Operators need control-plane actions for safe link management.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Demonstration
- **Parent:** L1-LNK-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN an enabled link WHEN disable operation is requested THEN link state changes to disabled and is reflected in subsequent status queries.

## L2-LNK-003

- **Requirement:** Tranquility shall expose documented link custom action execution behavior.
- **Rationale:** Missions use link actions for adapter-specific control operations.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Test
- **Parent:** L1-LNK-001
- **Priority:** Should
- **Acceptance Criteria:**
  - GIVEN a link with an advertised action WHEN run-action is invoked THEN action result is returned according to method contract.

## L2-LNK-004

- **Requirement:** Tranquility shall publish link in/out counters through the link information model.
- **Rationale:** Throughput trend analysis and outage diagnosis rely on link counters.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Inspection
- **Parent:** L1-LNK-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN active link traffic WHEN link status is queried THEN inbound and outbound counters are present and nondecreasing.
