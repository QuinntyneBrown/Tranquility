# CFDP subsystem — L2 requirements

## L2-FDP-001

- **Requirement:** Tranquility shall implement transfer creation at `POST /api/filetransfer/{instance}/{serviceName}/transfers` with documented request semantics.
- **Rationale:** Transfer creation is the entry point for CFDP API operations.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Test
- **Parent:** L1-FDP-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN a valid transfer request WHEN posted to the create-transfer URI THEN a transfer identifier and initial transfer state are returned.

## L2-FDP-002

- **Requirement:** Tranquility shall support transfer pause, resume, and cancel lifecycle actions through documented API methods.
- **Rationale:** Operators require runtime control of long-running transfers.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Demonstration
- **Parent:** L1-FDP-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN an active transfer WHEN pause/resume/cancel methods are invoked THEN transfer state transitions reflect requested operation.

## L2-FDP-003

- **Requirement:** Tranquility shall provide transfer status subscriptions via the documented file transfer subscription methods.
- **Rationale:** Clients need push-based monitoring of transfer progress and failure states.
- **Source:** External API documentation package (URL redacted)
- **Verification:** Test
- **Parent:** L1-FDP-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN an active transfer subscription WHEN transfer state changes THEN a status update is delivered on the subscription stream.

## L2-FDP-004

- **Requirement:** Tranquility shall document and enforce one baseline CFDP profile for conformance testing.
- **Rationale:** Differential testing requires a fixed, explicit protocol profile.
- **Source:** CCSDS 727.0-B (profile definition pending: TBD-008), User brief §8
- **Verification:** Inspection
- **Parent:** L1-FDP-001
- **Priority:** Should
- **Acceptance Criteria:**
  - GIVEN baseline conformance tests WHEN the profile declaration is reviewed THEN each test case references a single declared CFDP profile.
