# Security subsystem — L1 requirements

## L1-SEC-001

- **Requirement:** Tranquility shall enforce authentication and authorization for API access according to documented IAM behavior.
- **Rationale:** Mission operations require controlled access to telemetry, command, and configuration interfaces.

- **Verification:** Test

## L1-SEC-002

- **Requirement:** Tranquility shall provide TLS-protected transport and auditable security-relevant event records.
- **Rationale:** Secure transport and accountability are mandatory for command-and-control systems.
- **Source:** External API documentation package (URL redacted), External API documentation package (URL redacted), User brief §9
- **Verification:** Demonstration
- **Priority:** Must
