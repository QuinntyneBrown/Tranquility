# Space Data Link Protocol subsystem — L2 requirements

## L2-SDL-001

- **Requirement:** Tranquility shall accept TM transfer frames conforming to the selected CCSDS 132.0-B mission profile.
- **Rationale:** TM ingestion is a baseline telemetry entry path.
- **Source:** CCSDS 132.0-B (profile/clause mapping pending: TBD-002)
- **Verification:** Test
- **Parent:** L1-SDL-001
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN TM frames compliant with the mission profile WHEN ingested THEN frames are accepted for downstream packet extraction.

## L2-SDL-005

- **Requirement:** Tranquility shall generate an explicit validation diagnostic when a transfer frame violates the selected profile.
- **Rationale:** Operators need attributable failure reasons for operations and conformance testing.
- **Source:** User brief §9; CCSDS data-link standards (TBD-002..TBD-005)
- **Verification:** Demonstration
- **Parent:** L1-SDL-002
- **Priority:** Must
- **Acceptance Criteria:**
  - GIVEN a malformed transfer frame WHEN validation executes THEN an error diagnostic is recorded with failure reason and frame context.
