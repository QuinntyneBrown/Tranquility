# SDLP subsystem — L1 requirements

## L1-SDL-001

- **Requirement:** Tranquility shall process TM, AOS, USLP, and TC transfer frames using mission-selected CCSDS protocol profiles.
- **Rationale:** Data-link interoperability requires explicit profile conformance across supported frame families.
- **Source:** CCSDS 132.0-B, CCSDS 732.0-B, CCSDS 732.1-B, CCSDS 232.0-B (clause mappings pending: TBD-002..TBD-005)
- **Verification:** Test
- **Parent:** TRQ-HLR
- **Priority:** Must

## L1-SDL-002

- **Requirement:** Tranquility shall emit explicit diagnostics for transfer-frame validation failures.
- **Rationale:** Silent frame loss is unacceptable for operations and conformance triage.
- **Source:** CCSDS data-link standards (TBD-002..TBD-005), User brief §9 (tight error handling)
- **Verification:** Demonstration
- **Parent:** TRQ-HLR
- **Priority:** Must
