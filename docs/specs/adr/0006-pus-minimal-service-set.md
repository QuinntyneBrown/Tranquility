# ADR 0006: ECSS PUS minimal service set

- **Status:** OPEN
- **Date:** 2026-07-15
- **Related:** TBD-011

## Context

The baseline includes only a minimal ECSS PUS subset. The specific services and capability levels are mission-dependent and not yet fixed.

## Options

1. Select mission-tailored minimum set from ECSS-E-ST-70-41C service catalog.
2. Defer all PUS-specific service behavior to a later release.
3. Provide extension points in baseline and lock service set by mission ICD.

## Decision

OPEN — requires mission operations and systems engineering decision.

## Consequences to evaluate

- Interoperability with existing mission procedures.
- Scope containment for v1.
- Verification effort.
