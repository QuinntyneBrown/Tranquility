# ADR 0005: Performance target baseline

- **Status:** OPEN
- **Date:** 2026-07-15
- **Related:** TBD-013, L1-QLT-002

## Context

Baseline requirements mandate explicit performance targets, but numeric thresholds are not yet approved.

## Options

1. Set conservative initial SLOs, revise by benchmark evidence.
2. Derive SLOs from mission data-rate envelopes before implementation freeze.
3. Define tiered SLO classes by deployment profile.

## Decision

OPEN — requires product/performance approval.

## Consequences to evaluate

- Acceptance test pass/fail criteria.
- Capacity planning.
- Risk of late requirement churn.
