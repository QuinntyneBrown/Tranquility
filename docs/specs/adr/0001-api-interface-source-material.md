# ADR 0001: API interface source material

- **Status:** OPEN
- **Date:** 2026-07-15
- **Related:** TBD-012, TRQ-ICD-API

## Context

Tranquility requires wire compatibility with the documented reference mission-control system API while maintaining a clean-room process and Apache-2.0 distribution goals.

## Options

1. Derive interface only from published prose docs.
2. Use published `.proto` files as interface descriptions.
3. Publish a native Tranquility API and separate compatibility shim.
4. Negotiate a licensing arrangement with API owner.

## Decision

OPEN — pending product/legal decision.

## Consequences to evaluate

- Legal risk profile.
- Completeness and speed of compatibility delivery.
- Long-term maintenance burden.
